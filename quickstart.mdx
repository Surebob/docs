## Prerequisites

- Node.js 20+ installed
- WSL2 (for Windows users)
- Docker Desktop (for container support)

## Start the Backend

<Steps>
  <Step title="Clone and Install">
    ```bash
    cd /home/wgtgr/code-exec
    pnpm install
    ```
  </Step>

  <Step title="Start Development Server">
    ```bash
    cd packages/backend
    pnpm dev
    ```
    
    First run builds Docker container (~2-3 minutes)
    
    Subsequent runs are fast!
  </Step>

  <Step title="Verify Health">
    ```bash
    curl http://localhost:8787/health
    ```
    
    **Expected:**
    ```json
    {
      "status": "ok",
      "version": "3.0.0"
    }
    ```
  </Step>
</Steps>

## Execute Your First Code

### JavaScript (Isolate - Fast!)

```bash
curl -X POST http://localhost:8787/api/execute \
  -H "Content-Type: application/json" \
  -d '{
    "userId": "test-user",
    "code": "console.log(\"Hello from isolate!\"); console.log(\"2 + 2 =\", 2 + 2);",
    "runtime": "javascript"
  }'
```

**Response:**
```json
{
  "success": true,
  "output": "Hello from isolate!\n2 + 2 = 4",
  "logs": ["Hello from isolate!", "2 + 2 = 4"],
  "duration": 120
}
```

### TypeScript (Container - With Types!)

```bash
curl -X POST http://localhost:8787/api/execute \
  -H "Content-Type: application/json" \
  -d '{
    "userId": "test-user",
    "code": "interface User { name: string; } const user: User = { name: \"Test\" }; console.log(user.name);",
    "runtime": "typescript"
  }'
```

### Python (Container - Data Analysis!)

```bash
curl -X POST http://localhost:8787/api/execute \
  -H "Content-Type: application/json" \
  -d '{
    "userId": "test-user",
    "code": "import os\ntools = os.listdir(\"/mnt/servers/cosmo-tools\")\nprint(f\"Found {len(tools)} tools\")",
    "runtime": "python"
  }'
```

## Test Tool Discovery

### Discover MCP Tools (JavaScript)

```javascript
import fs from 'node:fs';

// Discover servers
const servers = fs.readdirSync('/mnt/servers');
console.log('Servers:', servers);

// Discover tools
const tools = fs.readdirSync('/mnt/servers/cosmo-tools');
console.log('Found', tools.length, 'tools');

// Read tool documentation
const code = fs.readFileSync('/mnt/servers/cosmo-tools/searchEmails.js', 'utf-8');
console.log('Tool code length:', code.length);
```
